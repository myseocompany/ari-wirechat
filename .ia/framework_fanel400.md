# Framework para LLM – Copy y Embudos de Curiosidad

Guía operativa para que un modelo de lenguaje genere todos los activos de comunicación inspirados en las 3 sesiones del entrenamiento **Funel 400%**. Usa esta referencia como “sistema operativo” antes de pedir piezas concretas (anuncios, quizzes, páginas, secuencias).

---

## 1. Datos que siempre debes alimentar al LLM

1. **Producto/servicio** (qué vende y transformación principal).  
2. **Cliente ideal** (dolores, contexto, lenguaje que usa).  
3. **Promesa núcleo** (resultado, tiempo, condiciones).  
4. **Pruebas o historias** disponibles.  
5. **Restricciones** (tono, canales, presupuesto).

Sin estos datos el modelo solo podrá entregar textos genéricos.

---

## 2. Principios que deben reflejarse en cada salida

### 2.1 Curiosidad Progresiva (Día 1)

Secuencia de nueve micro‑gatillos:

1. **Vacío de curiosidad**: abre con una pregunta o escena que expone un problema escondido (“Descubre qué tan…”, “¿Qué pasa si…”).  
2. **Espejo**: etiqueta a la persona con un concepto nuevo (“frontera dental”, “hijo lámpara”) para que se auto‑diagnostique.  
3. **Momento ajá**: explica por qué eso importa (ej. datos, comparación inesperada).  
4. **Menú de posibilidades**: muestra errores o caminos alternos.  
5. **Puente obvio**: introduce tu metodología único nombre: *Fanel 400%*, *cerebro perezoso*, etc.  
6. **Argumento**: cómo funciona (pasos o metáfora práctica).  
7. **Consecuencia**: qué pasa si no actúa.  
8. **Prueba breve**: ejemplo, historia corta, métrica.  
9. **Siguiente paso**: CTA específico.

### 2.2 Agujero Negro (Día 2)

Reemplaza “prometer más” por “explicar distinto”.

- Presenta un concepto inesperado que resignifique al producto (“Tocar piano es memorizar 6 números”).  
- Usa analogías visuales (clientes compitiendo frente al mismo prospecto, cerebro en un sillón, etc.).  
- Pregunta al LLM: *“¿Qué explicación nueva vuelve inevitable mi solución?”* y exige que la respuesta se pueda demostrar con una escena o gesto.

### 2.3 Fanel 400% – Dopamina inmediata (Día 3)

El embudo debe:

1. **Inflar el globo**: gatillos rápidos de curiosidad.  
2. **Darle escalones al cerebro**: pasos cortos, nunca saltos.  
3. **Generar posesión**: quizzes, preguntas, decisiones.  
4. **Crear sensación de avance**: cada interacción desbloquea algo (diagnóstico, puntaje, recomendación).  
5. **Hacer irrelevante a la competencia**: posiciona a la solución como la única que respeta esos principios.

---

## 3. Componentes que el LLM debe producir

> Cuando pidas activos, referencia explícitamente las secciones siguientes para que el modelo siga la estructura.

### 3.1 Gancho y narrativa principal

**Prompt base:**  
“Genera 5 ganchos que abran con vacío de curiosidad, nombren un ‘agujero negro’ propio y terminen en un CTA a [acción]. Usa lenguaje [tono], máximo 25 palabras.”

Cada gancho debe incluir: escena visual + etiqueta nueva + pregunta.

### 3.2 Scripts para Reels/Anuncios

Estructura:

1. **Escena inmediata** (inserta al prospecto en la situación).  
2. **Descripción exagerada/analogía** (cliente, cerebro, competencia).  
3. **Presenta el “fanél/quiz/concepto”** como agujero negro.  
4. **Argumenta con micro pasos** (qué estimula, por qué funciona).  
5. **Beneficio tangible** (compran 4x más rápido, etc.).  
6. **CTA** (evento, link, mensaje).

Pídele al LLM que entregue:

- Versión hablada (guion)  
- Lista de gestos/props sugeridos  
- Subtítulos listos para editar

### 3.3 Quiz/Diagnóstico de Curiosidad

Objetivo: que el prospecto participe y sienta que descubre algo propio.

Instrucciones para el LLM:

1. Diseña **5 a 7 preguntas** (máx. 3 opciones cada una).  
2. Mezcla preguntas funcionales + emocionales.  
3. Etiqueta resultados con nombres memorables.  
4. Incluye explicación corta para cada resultado: qué significa, error típico, primer paso recomendado.  
5. CTA final: invita a agendar, ver demo o descargar guía.

### 3.4 Página de resultados / Landing

Bloques mínimos:

1. **Headline de curiosidad** (dato o pregunta).  
2. **Radiografía del estado actual** (usa el nombre del resultado).  
3. **Agujero negro**: explicación fresca del problema.  
4. **Puente: Fanel 400%** (pasos + visual imaginario).  
5. **Prueba** (caso, métrica).  
6. **Oferta innegable** (qué recibe, plazos, soporte).  
7. **CTA escalonado** (diagnóstico + llamada/compra).

### 3.5 Secuencia de follow-up

Solicita al LLM:

1. **Email/WhatsApp 1** – Reavivar el momento ajá (anécdota).  
2. **Email 2** – Explicar el agujero negro con otra metáfora.  
3. **Email 3** – Caso de éxito + CTA.  
4. **Mensaje corto** – Recordatorio amigable con deadline.

---

## 4. Checklist para validar cada entrega

- ¿Existe un vacío de curiosidad claro en la primera línea?  
- ¿El prospecto recibe un nuevo lenguaje/etiqueta para describir su problema?  
- ¿Se explica por qué la solución es distinta (agujero negro)?  
- ¿Cada texto invita a una acción concreta y pequeña?  
- ¿La dopamina inmediata está presente (puntaje, avance, promesa de descubrimiento)?  
- ¿Se evita competir por precio o descuentos?

Pide al LLM que marque explícitamente dónde se cumple cada punto.

---

## 5. Cómo solicitar múltiples activos en un solo prompt

Ejemplo:

```
Usa el Framework Funel 400%.
Contexto: [producto/mercado].
Objetivo: llenar [evento/servicio] en [fecha].
Entrega:
1. 3 ganchos de curiosidad progresiva.
2. Script de reel (60s) con instrucciones visuales.
3. Quiz de 6 preguntas con 3 resultados nombrados creativamente.
4. Sección “Puente obvio” para la landing.
Incluye checklist de validación.
```

---

## 6. Recomendaciones finales para trabajar con el LLM

1. **Itera por bloques**: genera primero concepto y ganchos, luego anuncios, después quiz/página.  
2. **Re‑alimenta con métricas**: CTR, CPL, comentarios; pide nuevas versiones usando esos datos.  
3. **Obliga al modelo a justificar**: “explica por qué este copy genera dopamina inmediata”.  
4. **Solicita variantes** por avatar, canal y etapa (frío, tibio, caliente).  
5. **Mantén viva la curiosidad**: cada pieza debe dejar una pregunta abierta que la siguiente responda.

Con este documento puedes aprovisionar cualquier LLM para producir activos consistentes con las tres sesiones originales.
